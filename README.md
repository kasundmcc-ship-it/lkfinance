# LK Finance Pro — Collector App

Mobile-first PWA for daily loan collection.

## Setup
1. Upload all files in this folder to a GitHub repository
2. Go to Settings → Pages → Source: main branch / root
3. Your app URL will be: https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/
4. Share this URL with your collectors

## First-time collector setup
1. Open the URL on their phone
2. Log in with their name + password
3. Tap the URL bar → "Add to Home Screen"
4. They now have a one-tap app icon on their phone
5. In the "Me" tab, paste the Google Apps Script URL once
